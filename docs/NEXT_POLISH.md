# Next Polish Roadmap

This roadmap captures the next improvements that would make the bot and server feel more production-ready.

## High Impact

- Add Discord OAuth2 authentication to the dashboard.
- Add dashboard audit trail for every admin action.
- Add ticket priorities, status tags and staff notes.
- Add ticket reopen flow before deleting the channel.
- Add forum post templates for bugs, suggestions and reports.
- Add scheduled digest posts for open tickets, active bugs and top suggestions.

## Community Experience

- Add profile cards with XP, coins, CP, badges and creatures.
- Add weekly quests and limited-time shop rewards.
- Add seasonal badges and event roles.
- Add `/profile`, `/quests` and `/daily-streak`.
- Add welcome variations per language.
- Add language-specific announcement mirrors when needed.

## Moderation

- Add configurable punishment ladder.
- Add case IDs for every moderation action.
- Add `/case`, `/cases` and `/reason` commands.
- Add log filters in the dashboard by event type.
- Add AutoMod review queue for deleted/blocked content.

## Open Source Quality

- Add GitHub Actions CI.
- Add `.env.example` validation.
- Add unit tests for ticket modal and transcript generation.
- Add architecture diagrams.
- Add issue templates for bug reports and feature requests.
